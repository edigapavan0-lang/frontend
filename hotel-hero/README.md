# Hotel Hero Images

This directory should contain background images of famous hotels in India for the hotel search form.

## Required Images

Please add the following images to this directory:

1. `taj-mahal-palace.jpg` - The Taj Mahal Palace, Mumbai
2. `oberoi-udaivilas.jpg` - The Oberoi Udaivilas, Udaipur
3. `raja-palace.jpg` - Umaid Bhawan Palace, Jodhpur or similar palace hotel
4. `leela-palace.jpg` - The Leela Palace, Bangalore/Delhi
5. `itc-maurya.jpg` - ITC Maurya, New Delhi
6. `amarvilas.jpg` - The Oberoi Amarvilas, Agra

## Image Specifications

- Format: JPG or JPEG
- Recommended size: 1920x1080px or larger
- Aspect ratio: 16:9 (landscape)
- File size: Optimized for web (under 500KB per image recommended)

## Usage

These images are used as rotating backgrounds for the hotel search form, similar to the flight search form.
